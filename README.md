News 24/7

News 24/7 is a React-based web application that aggregates news from various sources and displays them in a single, convenient location. The app fetches the latest news articles from different websites using their APIs and presents them in an organized manner, allowing users to stay up-to-date with current events effortlessly.

Features
Aggregated News: Fetches news from multiple sources and displays them in one place.
Categorized News: News articles are categorized for easy navigation (e.g., technology, sports, politics).
Responsive Design: Works seamlessly on both desktop and mobile devices.

Technologies Used
React: Front-end library for building user interfaces.
React Router: For handling navigation within the app.
Bootstrap: CSS framework for responsive design.
News APIs: To fetch news articles from various sources (e.g., NewsAPI, Guardian API)
